Variables y Constantes

Fundamentos de Python

En Python, una variable es un espacio de memoria que almacena un valor que puede cambiar durante la ejecución del programa. Se crean simplemente asignando un valor a un nombre utilizando el signo igual (=).

Una constante es un valor que no cambia a lo largo de la ejecución del programa. Aunque Python no tiene soporte directo para constantes, por convención, los nombres de las constantes se escriben en mayúsculas.

Tipos de Datos

Python tiene varios tipos de datos básicos:

  • Enteros (int): números enteros, e.g., 1, 42, -5.
  • Punto flotante (float): números decimales, e.g., 3.14, 2.0.
  • Cadenas de texto (str): secuencias de caracteres, e.g., "Hola", 'Python'.
  • Booleanos (bool): valores de verdadero o falso, e.g., True, False.
  • Listas (list): colecciones ordenadas de elementos, e.g., [1, 2, 3].
  • Tuplas (tuple): colecciones ordenadas e inmutables, e.g., (1, 2, 3).
  • Diccionarios (dict): colecciones de pares clave-valor, e.g., {"nombre": "Juan", "edad": 30}.

Condicionales

Los condicionales permiten tomar decisiones en el código según ciertas condiciones. La estructura básica es if, elif y else.

Ciclos

Los ciclos permiten repetir un bloque de código varias veces. Los dos tipos principales son for y while.

  • Ciclo for: se usa para iterar sobre una secuencia (como una lista o una cadena).
  • Ciclo while: repite un bloque de código mientras una condición sea verdadera.

Funciones

Las funciones son bloques de código que realizan una tarea específica y pueden ser reutilizados. Se definen usando la palabra clave def.

Estructuras de Datos Básicas

  • Listas (list): colecciones ordenadas y mutables.
  • Tuplas (tuple): colecciones ordenadas e inmutables.
  • Diccionarios (dict): colecciones de pares clave-valor.
